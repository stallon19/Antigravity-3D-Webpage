How to Use the AntiGravity Landing Page Template
1. Download and Extract the Files
After purchasing the template, download the ZIP package and extract it on your computer.

You will see the following folder structure:

antigravity-template/
│
├── index.html
├── assets/
│   ├── css/
│   ├── js/
│   ├── images/
│   └── fonts/
├── documentation/
└── README.txt


2. Open the Template
To view the landing page:

Navigate to the project folder.
Locate the file index.html.
Double-click the file or open it using any modern browser.
Supported browsers:

Google Chrome
Mozilla Firefox
Microsoft Edge
Safari
The landing page will load instantly in your browser.

3. Edit the Content
To modify the website content:

Open the project folder in a code editor like Visual Studio Code.
Open the file:

index.html

Edit sections such as:
Headings
Paragraph text
Buttons
Links
Images
Example:

<h1>AntiGravity</h1>
<p>Your futuristic landing page</p>


You can replace the text with your own content.

4. Change Images
All images are located in the images folder.

assets/images/



Steps:

Replace the existing image with your new image.
Use the same file name or
Update the image path in index.html.
Example:

<img src="assets/images/hero-image.png">


5. Customize Colors and Styles
To change the appearance of the template:

Open the stylesheet file:
assets/css/style.css

Modify colors, fonts, spacing, or animations.
Example:

body {
background: #0a0a0a;
color: white;
}


6. Modify Animations and Effects
Interactive animations are controlled using JavaScript.

Open:

assets/js/script.js

You can adjust:

Animation speed
Scroll effects
Interactive elements
7. Upload to Your Website
After customization, upload the files to your hosting server.

You can upload using:

FTP client like FileZilla

or

Hosting control panels like cPanel.

Upload all files while keeping the folder structure unchanged.

8. Test the Website
After uploading:

Visit your website URL.
Check if all images, styles, and scripts load correctly.
Verify responsiveness on mobile devices.
9. Browser Compatibility
This template is tested and works on modern browsers including:

Chrome
Firefox
Edge
Safari
10. Support
If you need help customizing the template, you can contact the author through the Codester profile support section.
